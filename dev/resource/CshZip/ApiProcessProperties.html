Custom Process Properties

Any Apex class representing a custom process will have these properties populated prior to execution.

Parameters
Type: String
Serialized JSON of the Map<String,Object> representing the incoming message parameters.
StepConfig
Type: SObject
Step-specific configuration according to the same type as specified in the ESB Marker Class
AllowsCallouts
Type: Boolean
True when handling callouts on the first invocation, false when handling the main logic on the second invocation.
