Message Parameters

esb__Id
Type: Id
Entity Id of an SObject handed into a Sequence by a Job constructed using a Simple Job.
esb__SequenceName
Type: String
Routes a message to the first step of the sequence named here. This value is case-sensitive.
esb__EnqueueUserId
Type: Id
Read-only Id of the user that spawned the message (usually by enqueueing an esb.Job).
esb__GateItemParameters
Type: List<Map<String,Object>>
Collection of message parameters from all of the gated messages.
