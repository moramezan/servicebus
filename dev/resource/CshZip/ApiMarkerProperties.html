Marker Properties

Any Marker Class must contain at minimum a Description property. Other properties exist to define the behaviour of any given process.

Description
Type: String
Required short description that explains what this process does.
Tag
Type: String
Optional grouping tag that can be used to align related processes together.
Name
Type: String
Optional friendly name to be used instead of the fully-qualified Apex Class name.
Icon
Type: String
Optional icon name from the Silk library that will display alongside steps of this process type.
HelpUrl
Type: String
Optional Help link that will be displayed alongside steps of this process type.
Limits
Type: Integer
Optional number of times this process can run within the governor limits.
HelpUrl
Type: Integer
Optional number of times this process can run within the governor limits.
Inputs
Type: Map<String,String>
Optional friendly descriptions of the input parameters consumed by this process.
Outputs
Type: Map<String,String>
Optional friendly descriptions of the output parameters emitted by this process.
ProcessSetting
Type: SObject
Optional Custom Setting that can be used to link to global, Process-wide settings from the Sequence Builder
StepConfig
Type: SObject
Optional Custom Object that can be used to expose local, Step-specific configs from the Sequence Builder.
