esb.Job Constructors

All work in the ESB framework is initiated by invoking System.enqueueJob on an esb.Job which can use any of the following constructors:

Single Message Job

global esb.Job(Map<String,Object> messageParameters)

messageParameters
Type: Map<String,Object>
A parameterized description of the message data, which must at minimum contain esb__SequenceName.

Multiple Message Job

global esb.Job(List<Map<String,Object>> messageParametersList)

messageParametersList
Type: List<Map<String,Object>>
A collection of multiple parameterized message descriptions, each of which must at minimum contain esb__SequenceName.

Simple Job

global esb.Job(String sequenceName, Id entityId)

sequenceName
Type: String
Name of the sequence the data is destined for. This sets the esb__SequenceName message parameter for convenience.
entityId
Type: Id
Id of some record that the sequence will operate on. This sets the esb__Id message parameter for convenience.
