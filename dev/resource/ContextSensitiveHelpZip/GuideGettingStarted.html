Getting Started

The list below describes the order of activities for setting up and using your Services framework with references to other parts of this guide.

System Installation

  • Install the Service package.
  • The package self-configures. If however you are unable to complete the basic tutorials linked below, head over to the installed packages tab in your org and click the Configure link on the Service package listing.
  • Assign Orchestrator permissions to specified user(s).
  • Confirm that the system has been correctly installed by completing the Hello World Tutorials.

Application Design and Configuration

  • Design your messaging application with reference to the business specification (requirements)
  • Define appropriate inbound events, and the injection of those events into the Services framework.
  • Design your own services as needed. This topic is described in the detail the separate Service Developers Guide.
  • Configure sequences and processing steps as needed. For more details, refer to the Sequences help.
  • Monitor the system using the Wiretap service and Messages tab. For further inspiration complete the Valis Salesforce Ids tutorial
