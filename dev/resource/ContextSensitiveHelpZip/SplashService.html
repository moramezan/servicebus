Services

What is a service?

A service is a single piece of business logic implemented as an Apex class. Every service handles one event and emits a collection containing zero, one or many new events. Each service class contains a Service Definition that describes its behavior.

How to enqueue events to a service?

event = new Map<String,Object>();
event.put('ToService', 'MyServiceName');
Services.Enqueue.invoke(event);

How to author custom services? 

global class MyService extends Abstract.Service {
    
    global class Definition extends Abstract.Service.Definition {
        String Description = 'Short explanation of the service.';
    }
    
    override public void callout(Map<String,Object> event) {
        //perform service callouts
    }
    
    override public List<Map<String,Object>> execute(Map<String,Object> inputEvent) {
        //implement service logic
    }
    
}
