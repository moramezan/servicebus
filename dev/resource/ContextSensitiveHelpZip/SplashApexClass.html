Processes

What is a process?

A process is a single, logical program with inputs and outputs in the form of events. Processes are implemented as Apex classes. Each process handles one event, and may output zero, one, or many new events.

  • Each process has a Summary that describes it.
  • An event is represented in Apex code as Map<String,Object>
