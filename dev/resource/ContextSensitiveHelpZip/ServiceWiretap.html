Wiretap Service

The Wiretap service will send you an email containing all available input event parameters seen by its step in a Sequence.

Note Wiretap is a 'pass-through' service which does not alter the event parameters or have any other side effects. It is primarily a learning and debugging tool.

Settings

The email will be sent to the user under which the Service Bus is running. Alternatively, a specific user can be nominated to receive these emails.

To specify the Wiretap recipient:

  1. Open the Services tab, then Go to view all services.
  2. Click on the Wiretap service, then click View Settings. This will take you to the Wiretap Custom Settings view.
  3. Here you can create an organization-wide setting called User Id that specifies the user to whom Wiretap emails will be sent. Enter the Id of the user you nominate.
