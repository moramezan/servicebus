Post-build steps

Visually test the service

Once the Apex class has successfully compiled, it can be registered in the Service Bus. To register an Apex class in the Service Bus, select the list view on Processes tab to reveal the Discover button.

  • Click Discover and your new Apex class will present itself after a few minutes in the list of available processes. (For further details on the generation process, see the Process Discovery Tutorial.
  • Create a suitable sequence to test your processes by visiting the Sequences tab. (For further details on creating and testing sequences, see the Setup Confirmation Tutorial.

Creating a managed package containing your service

You can create a managed package managed package by following standard Salesforce recommendations.

Add your Apex class to the package along with unit tests and associated dependencies and upload.

This compiled process logic can now be installed in any Developer, Sandbox or Production org using the the package link.
