Static Enqueue

Single event invoke

Sends an event with any number of event parameters to a sequence or a service.

global Id invoke(Map<String,Object> event)

event
Type: Map<String,Object>
An event, which must carry a ToService or ToSequence string parameter.

Multiple event invoke

Sends multiple events with any number of event parameters to sequences or services.
Use this constructor to bulkify multiple events instead of calling invoke inside of a loop.

global Id invoke(List<Map<String,Object>> events)

events
Type: List<Map<String,Object>>
A collection of multiple events, each of which must carry a ToService or ToSequence parameter with a string value.

Invocable Action

Process Builder invoke

Lightning Processes can leverage the Services Package to do asynchronous heavy lifting from Process Builder.

global Id invoke(List<SimpleEvent> events) 

    Services.Enqueue.SimpleEvent {
        String RecordId;
        String ToService;
        String ToSequence;
        String CorrelationUuid;
    }
RecordId
Type: Id
Indicates which sObject the sequence will operate on.
ToService
Type: String
Indicates which service the event is destined for.
ToSequence
Type: String
Indicates which sequence the event is destined for.
CorrelationUuid
Type: String
Universally unique identifier (v4) which can be used to group everything spawned from any entry point event.
