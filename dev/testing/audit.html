Operating the audit processes (DO NOT PERFORM THIS TEST)

The audit processes store event parameters so they can be viewed later.
Normally this is used for debugging, but they are available on Message__c too.

Procedure

  1. Create a new Sequence 'AuditTest' containing an 'Audit Initializer' step in position 1.
  2. Add three new 'Audit Writer' steps to the sequence. The 'Terminate' should be at the end.
  3. Click Enqueue Test Event.
  4. Wait for the monitor to show all processing has finished.

Outcomes

  1. No messages should be graphed in the monitor.
  2. View the 'AuditTest' sequence and click Configure on the 'Audit Initializer' step.
  3. The Audit Groups listview should list exactly one record.
