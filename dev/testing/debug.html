Operating the debug process

The debug process reads all event parameters and emails them to the user.
The user is nominated through the Debug custom setting.

Procedure

  1. From Setup > Custom Settings > Debug Settings, create an org level setting with your User Id.
  2. Create a new Sequence 'DebugTest' containing a 'Debug' step in position 1.
  3. Click Enqueue Test Event.
  4. Wait for the monitor to show all processing has finished.

Outcomes

  1. No messages should be graphed in the monitor.
  2. The monitor should graph exactly one 'Paused' message for 'Debug'.
