Pausing the message queue

Pause will temporarily stops the broker. New messages accumulate in an unprocessed state.
This is used to ensure safe downtime while modifying a Sequence.

Procedure

  1. From any Messages listview, click Pause Queue.
  2. Create a new Sequence containing a 'Wiretap' step in position 1.
  3. Click Enqueue Test Message

Outcomes

  1. The monitor should graph exactly one 'Paused' message for 'Wiretap'.
  2. Broker should not be running. The red twirly should be absent.
